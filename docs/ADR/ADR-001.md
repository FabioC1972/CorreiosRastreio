# ADR-001

## Título

Escolha do Next.js para o Correios Rastreio

## Status

Aprovado

## Data

2026-07-21

## Responsáveis

Fabio Carraro
Yuri

## Contexto
O projeto Correios Rastreio precisava de uma tecnologia moderna, rápida, otimizada para SEO e de fácil hospedagem. Como objetivo, busca oferecer um portal de rastreamento de encomendas com excelente desempenho e capacidade de expansão futura.

## Decisão
Foi escolhido o framework Next.js como base do projeto Correios Rastreio por oferecer renderização otimizada para SEO, excelente desempenho, facilidade de hospedagem na Vercel e uma arquitetura moderna baseada em React.

## Consequências
O projeto passa a utilizar oficialmente o Next.js como framework principal. Todas as novas funcionalidades deverão seguir essa decisão, salvo a criação de um novo ADR que a substitua.

## Justificativa
A escolha do Next.js foi baseada na necessidade de criar um projeto escalável, com excelente indexação pelos mecanismos de busca (SEO), boa experiência para o usuário, facilidade de implantação e grande adoção pela comunidade React.

## Alternativas Consideradas
- React puro
- Vue.js
- Angular
Após análise técnica, o Next.js apresentou melhor equilíbrio entre SEO, desempenho, produtividade e facilidade de implantação.
