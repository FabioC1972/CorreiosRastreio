# Correio Rastreio

Projeto completo em Next.js App Router, TypeScript e Tailwind CSS.

## Rodar localmente

```bash
npm install
npm run dev
```

Acesse `http://localhost:3000`.

## Códigos de teste

- `AA123456785BR` — entregue
- `BB987654321BR` — em trânsito
- `CC111222333BR` — fiscalização aduaneira

## Próximos passos

Substitua os mocks em `data/tracking.ts` por uma integração de API, os links de afiliado pelos links reais e os componentes `BannerAd` pelos blocos do Google AdSense.
